# Setup


## Software 

  - **Stata**. It is feasible to do the whole exercise in **R** as well, but examples will be in Stata.
  - **text editor** (we suggest [VS Code](https://code.visualstudio.com/))

## Setting up a project

1. Create an empty folder, like `myprojects/day1`
2. Navigate to this folder in Stata: `cd ../myprojects/day1`
3. (optional) Open this folder in your text editor, if it can open folders 

## Why a Text Editor?

- Allows to edit code and documentation in the same place
- Sometimes allows you to run code directly from the editor (**R**, with some limitation **Stata**)
- My preferred: 
  - [VS Code](https://code.visualstudio.com/)
  - manually run code from terminal (works on Mac, Linux, with some caveats Windows)

## Why terminal running?

- Terminal allows us to immediately be in the same environment that **push-button reproducibility** will use.
- Allows to test code repeatedly, easily.
